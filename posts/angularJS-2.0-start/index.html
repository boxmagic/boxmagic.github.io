Angular JS 2.0 簡介與環境建置

發表於 2017-05-13 | 分類於 Angular JS 2.0 |

簡介

AngularJS 是一款來自 Google 的前端 JavaScript 框架，也是 SPA ( Single Page-Application，單頁應用 ) 框架。它的目標是透過 MVC 模式 ( MVC ) 功能增強瀏覽器的應用，使開發和測試變得更加容易。AngularJS 框架的體積很小，但設計理念和功能卻非常強大，極大的簡化前端開發負擔，快速成為了 JavScript 的主流架構，幫助開發者從事 web 開發。
