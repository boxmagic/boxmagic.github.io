VS Code 提升產能的擴充套件

發表於 2017-04-09 | 分類於 VSCode |

前言

之前使用 VS Code 開發 Angular 2 時，當頁面越來越多，目錄越來越深，在尋找檔案或者是目錄時，變得不是很直觀，降低產能。

就上圖所示，檔案一多會變得不好辨認。
所以在檔案前面以圖示顯示，會比辨認副檔名還來得直觀，來得快。

安裝

  1. 安裝 vscode-icon 套件
    開啟 VS Code 左側有擴充功能圖示，點擊之後查詢「vscode-icon」，點擊安裝

    完成安裝後，上方設定列點擊「Code」> 「喜好設定」>「檔案圖示佈景主題」

    此時在 Code 應用程式上會出現選項，選擇「VSCode Icons」

    最後可以在左側目錄列看到每個檔案，包含資料夾都有圖示去辨別檔案的類型
