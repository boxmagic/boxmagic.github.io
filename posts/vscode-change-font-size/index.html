如何調整 VS Code 字體大小

發表於 2017-04-26 | 分類於 VSCode |

實作

在 VS Code 上，文件 > 選項 > 設定，就可以看到 settings.json，找到 「editor.fontSize」，將數字改掉即可在編輯視窗看見字體大小的變化。
