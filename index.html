IIS 遇到 「HANDLER "EXTENSIONLESSURLHANDLER-INTEGRATED-4.0" HAS A BAD MODULE "MANAGEDPIPELINEHANDLER" IN ITS MODULE LIST」 問題解決方式

發表於 2017-04-17 | 分類於 IIS ， Error |

前言

在建置 IIS 時，執行起來遇到 「HANDLER “EXTENSIONLESSURLHANDLER-INTEGRATED-4.0” HAS A BAD MODULE “MANAGEDPIPELINEHANDLER” IN ITS MODULE LIST」問題，可透過以下方式解決

實作

開啟控制台 > 新增或移除程式 > 程式和功能
將 Internet Information Service > World Wide Web Service > Application Develpoer 底下

  • .NET Extensibility 4.5
  • ASP.NET 4.5

這兩項打勾後安裝，即可將此問題解決
