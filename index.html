SQL Server 快速建立表

說明

最近新增多張表，要寫欄位名稱、欄位說明、設定索引、是否為增量值、預設值 … 等等，但是要一張一張表去新增，還要建立彼此的關聯，有點繁瑣與複雜，此時可以利用 SQL Server 的資料庫圖表來一次將這些資訊建立完成，並且可以自訂欄位的屬性，只要對資料庫圖表點右鍵 > 表規劃 > 修改自定義，就可以將欄位要編輯的屬性一次加入，方便開發人員利用此工具快速建立資料表，節省時間。
